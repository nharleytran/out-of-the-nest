# OOSE Project

# Install the app

We first need to install MongoDB, and NodeJs. Then we install packages in server and client directories.

```
   $ cd client
   $ npm install
   $ cd ..
   $ cd server
   $ npm install
   
```
To run the server,
```
$ cd server
$ nodemon index.js
```
and the client side,

```
$ cd client
$ npm start
```
